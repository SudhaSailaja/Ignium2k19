ESCAPE ROOMS



Escape rooms transports you into a realm of reasoning where you scramble though pictures and swim through a sea of code with every drop of stressed out grey matter, to set yourself free.

Rules



ELIGIBILITY : This event is open for all the departments.

EVENT DESCRIPTION:

Room 1 :
GEARING UP :
A test which includes aptitude, verbal ability, reasoning, logo guessing, tagline identification, cryptography and programming questions.The qualified teams get the key to escape to the next level.

Room 2 :
GOOGLING :
4 images will be given to qualified participants ,they have to google it and give the details of those images. The team which cracks the details within less time will be qualified to next round. If none of them cracks, certain hints will be released to them.

Room 3 :
BLIND CODING :
A coding problem will be given to the participants. While typing the code, monitor will be turned off. The teams with correct output or less number of errors will be qualified to next round.

Room 4 :
CLUELESS :
A jigsaw puzzle consisting of pieces representing various colors.There will be numbered clues in order whose output is the colour which has to be placed in that numbered position.The team that finishes first accurately wins.
