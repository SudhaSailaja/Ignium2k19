Rules



ELIGIBILITY :

This event is open to all regular B. Tech Students who are interested in Coding.


EVENT DESCRIPTION :
Round 1 : Team will be shown combination of pictures , logos and few puzzles which are to be solved in a given time.

Round 2 : Questions of logic gate, puzzles, logical reasoning will be given and are to be solved in given time. On solving these questions you get numbers. By using these numbers the participants have to form a path from start node to end node which solves the maze given.

Round 3 : Initially participants are given with some virtual points. Each question is assigned with some points. On displaying the question, participant who bids the question first will get the chance to solve the question. The one who solves more number of questions will be the winner.
